Step 2- Fork it into your GitHub account 

Step 3 - Click Code → Codespaces → Create codespace on main

Step 4 - Use the Codespaces extension in local VS Code

Step 5 - Activate a virtual environment 

Step 6 - Install packages 

Step 7 - Update requirements.txt 

Step 8 - Update GitHub
